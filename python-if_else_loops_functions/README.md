# Python - if/else, loops, functions

##   What the project does
This project introduces the essential building blocks of Python control flow and function usage.  
It covers indentation rules, conditional statements (`if`, `elif`, `else`), comments, variable assignment, loops (`while`, `for`), loop controls (`break`, `continue`, `else` on loops), the `pass` statement, the `range()` function, how functions work, what they return, variable scope, tracebacks, and arithmetic operators.

These concepts form the foundation for writing correct, readable, and efficient Python programs.

---

##   Why the project is useful
Understanding these topics is critical because:

- Python relies on **indentation** to define code blocks.
- Conditionals and loops allow you to **control program flow**.
- Functions help you **organize and reuse code**.
- Knowing how variables behave in different scopes prevents bugs.
- Recognizing tracebacks helps you **debug errors quickly**.
- Arithmetic operators and `range()` are used constantly in real programs.

Mastering these basics prepares you for more advanced Python topics and real‑world software development.

---

##   How users can get started with the project
To begin practicing the concepts in this project:

1. Install Python 3 on your system.  
2. Create a `.py` file and experiment with:
   - Writing `if/else` statements  
   - Creating loops with `while` and `for`  
   - Using `break`, `continue`, and `pass`  
   - Defining functions with `def`  
   - Testing variable scope inside and outside functions  
   - Triggering and reading tracebacks  
3. Run your code using:
   ```bash
   python3 filename.py
   ```

Start small, test often, and build confidence with each concept.
